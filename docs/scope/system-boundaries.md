# System Boundaries

## External Boundary

- Frontend SPA and external API consumers.
- Public blockchain RPC providers.
- Open market data providers.

## Internal Boundary

- Kong gateway and microservices network.
- Core ledger, AI, fraud, KYC, analytics, payment modules.
- Shared security middleware and observability services.

## Data Boundary

- Transaction and identity data stores.
- Event sourcing and immutable audit records.
